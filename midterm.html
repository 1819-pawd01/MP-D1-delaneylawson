Tables

HTML element table
element name element description element example
doctype html tag This is the first thing that should be in your HTML document <!doctype html>
comment tag this is how you add a comment to your HTML without it showing up on yur website <!--comment-->
p tag this tag is where your paragraph content goes <p>
h1 tag This tag is for your header content <h1>
line break tag this tag adds a space between two lines <br>
title tag the title of the document <title>
body tag The content for your whole document <body>
main tag this tag represents teh dominant content of the body of a document <main>
blockquote tag Indicates that whatever is in this tag is in italics and is a quote <blockquote>
table tag all the information for a table goes in this tag <table>
list tag any kind of list like unordered and ordered show by numbers or bullet points <li>
ordered list tag represent a list that is numbered <ol>
unordered list tag represented a list that has bullet points <ul>
horizontal line tag represents when there is a line separating two things <hr>
image tag this tag is the opening tag for an image <img>
